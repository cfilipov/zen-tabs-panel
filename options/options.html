Zen Tabs Panel

Auto-close unpinned tabs

Automatically close tabs that haven't been visited within the configured time. Pinned tabs and Essentials are never closed.

Auto-move active tab to top

Move the current tab to the top of the tab bar when activated. Only applies to unpinned tabs.

Extension popup windows

Catch browser.windows.create({type: "popup"}) calls from any installed extension (e.g. Bitwarden's biometric prompt or "Pop out into new window") and host the popup inside this extension's overlay instead of letting Zen open a real OS window. In macOS fullscreen, real popup windows cause a Spaces switch — this keeps them in-place.

Companion Zen Mods

Optional Zen Mods installed alongside this extension. Each can be enabled or disabled independently from Zen's Mods settings.

Keyboard shortcuts

Customize shortcuts in about:addons › Extensions › Gear icon › Manage Extension Shortcuts.
