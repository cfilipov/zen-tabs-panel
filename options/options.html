Zen Tabs Panel

Auto-close unpinned tabs

Automatically close tabs that haven't been visited within the configured time. Pinned tabs and Essentials are never closed.

Auto-move active tab to top

Move the current tab to the top of the tab bar when activated. Only applies to unpinned tabs.

Extension popup windows

Catch browser.windows.create({type: "popup"}) calls from any installed extension (e.g. Bitwarden's biometric prompt or "Pop out into new window") and host the popup inside this extension's overlay instead of letting Zen open a real OS window. In macOS fullscreen, real popup windows cause a Spaces switch — this keeps them in-place.

Animations

When the palette opens or closes, fade/scale it in for a smoother appearance. Disabling this makes the palette appear and disappear instantly — snappier but more abrupt. Sub-menu transitions inside an open palette still animate either way.

Backdrop

Dim the content behind the palette while it's open. Disabling this leaves the underlying page fully visible; clicking outside the panel still dismisses the palette.

Chord delay

How long the chord engine waits after each key (in milliseconds) before opening the next menu. Applies to the leader pause (e.g. cmd+. alone), prefix pause (e.g. cmd+., o alone), and open-view pause (e.g. cmd+., q alone). Lower values feel snappier; higher values give you more time to think between keys. Set to 0 to show the menu immediately on every chord key — the menu effectively becomes the chord progress indicator.

Companion Zen Mods

Optional Zen Mods installed alongside this extension. Each can be enabled or disabled independently from Zen's Mods settings.
